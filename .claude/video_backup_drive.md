# MaccabiPedia Video Backup Drive

**Location on Maccabipedia drive:** `מכביפדיה_ראשי/וידאו`

## Structure

```
וידאו/
├── כדורגל/
│   ├── עונות/
│   │   └── <YYYY-YY>/               # one folder per season, e.g. 2016-17
│   │       └── <competition>/       # ליגה לאומית / גביע המדינה / גביע הטוטו / אירופה / ...
│   │           └── <stage or round>/  # e.g. מחזור 12 / גמר / חצי גמר
│   │               └── <video file>
│   ├── שחקנים/                      # player highlight compilations
│   └── מיוחדים/                     # special moments / general compilations
├── כדורסל/
│   └── <YYYY-YY>/
│       └── <video file>
└── כדורעף/
    └── <video file>                 # flat, no sub-folders
```

## Notes

- Seasons span 1935-36 through 2025-26; coverage is sparse for early seasons and dense around 2013-2017.
